# Allison Gamble Personal Website

![Website Figma Design](../assets/full_design_photo_1.png)


## Header

The header is a purple bar as in the photo above.  The text "allison gamble" on the left.  A box with "jackpot" on the right outlined with a rounded rectangle.

## Slot Machine

The slot machine is in the middle of the page, slightly overlapping the header bar above.  The jackpot should automatically resize based on the browser window size.  Whenever you pull the lever, an animation of a slot machine runs.  You can pull the lever repeatedely until you hit a jackpot.  Once a jackpot is hit, an animation evolves in the page celebrating the jackpot.
